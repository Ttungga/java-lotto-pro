# 로또
## 3단계 미션 기능 목록
* 구매 금액을 입력받는다.
  * 구매 금액 객체 생성
  * 입력값 유효성 검증
  * 구입 가능한 로또 티켓 수량 계산


* 로또 티켓을 생성한다.
  * 로또 번호들 객체 생성
    * 입력값 유효성 검증
  * 로또 티켓 객체 생성
  * 자동 로또 번호 발급기 객체 생성
  * 로또 티켓들 객체 생성


* 당첨 번호를 입력받는다.


* 로또 티켓별 상금을 판별한다.
  * 상금 유형 객체를 생성한다.
  * 로또 티켓별 어떤 상금 유형을 가지는지 확인한다.


* 당첨 결과를 출력한다.
  * 상금 유형별 통계 계산
  * 수익률 계산


* 입력 및 결과 안내 메시지를 출력한다.

## 4단계 미션 기능 목록

* 보너스 볼을 입력받는다.
  * 보너스 볼 객체 생성


* 상금 유형에 2등(5개 일치 + 보너스 볼 일치)을 추가한다.
  * 2등 상금 유형에 대한 당첨 여부를 확인한다.

## 5단계 미션 기능 목록

* 수동으로 구매할 수량을 입력받는다.
  * 수동 구매 수량 유효성 검증


* 수동으로 구매할 로또 번호를 입력받는다.
  * 수동 구매 로또 번호 유효성 검증


* 수동으로 구매한 티켓과 자동으로 발급한 티켓의 결과를 합산한다.


* 모든 티켓의 당첨 결과를 보여준다.
